Conexión Firebase (Nube)

Conecta una vez y todos tus dispositivos se sincronizan automáticamente. Sin Firebase, la app funciona solo en este equipo.

Sin configurar — solo modo local activo.
Ingresar credenciales de Firebase
Cómo obtener las credenciales:
1. Ve a console.firebase.google.com
2. Crea un proyecto → Agrega una app Web (</>)
3. Copia el objeto firebaseConfig que aparece
4. Ve a Firestore Database → Crear base de datos → Modo de prueba
5. Pega cada valor en los campos de abajo

Si ya tienes datos guardados en este equipo, súbelos a la nube:

Reglas de Firestore vencidas
El modo de prueba de Firebase dura solo 30 días. Para renovarlo:
  1. Ve a console.firebase.google.com
  2. Tu proyecto → Firestore Database → Reglas
  3. Cambia la fecha: allow read, write: if request.time < timestamp.date(2026, 12, 31);
  4. Haz clic en Publicar
